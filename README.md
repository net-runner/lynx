## Tech Stack
- **Database**: PostgresSQL & Prisma as a ORM
- **Frontend**: Next.js
- **Backend**: Node.js + Hyper-Express

## Features

- Dark mode from the start
- Link sharing, discovery, managing, observing link groups or users
- Clean, modern design
- Fast, and ready as PWA
